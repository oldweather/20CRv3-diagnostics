20CRv3 Diagnostics
==================

This is a set of analyses of the not-yet-released Twentieth Century Reanalysis version 3 (20CR3). It compares 20CRv3 to `20CR version 2c <https://www.esrl.noaa.gov/psd/data/20thC_Rean/>`_, to `CERA-20C <https://www.ecmwf.int/en/forecasts/datasets/archive-datasets/reanalysis-datasets/cera-20c>`_, and to `newly-digitised observations from around the British Isles <https://oldweather.github.io/DWR/>`_.
